## Fricções e soluções
- **DSN Invisível:** O `.env` central não tinha `IMPORTER_DB_DSN` e o banco do importer é `sharebook_importer`. Solução: Investigação via script Python para listar databases e construção manual do DSN.
- **Paths de Container vs Local:** O importer tentava rodar scripts em `/data/workspace/...`. Solução: Mapeamento manual no `.env` local.
- **Encoding de Console Windows:** Ocorreu `UnicodeDecodeError` ao capturar output de scripts. Solução: Injeção de `PYTHONIOENCODING=utf-8` no ambiente.
