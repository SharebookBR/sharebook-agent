## Decisões tomadas
- **Glob no Windows**: sempre usar caminho absoluto no `pattern`, nunca `path` + padrão relativo.
- **Editorial prompt no banco**: fonte da verdade editorial por source é `importer.sources.editorial_prompt`, não skill file.
- **Crawler direto no banco**: sem necessidade de arquivo de missão intermediário — inserção direta em `queue_items` como `waiting_triage`.
- **Nome da source**: `baixelivros_quadrinhos` (padrão snake_case + prefixo da fonte).

## Fricções e soluções
- **Glob path relativo não funcionou**: documentado na skill como armadilha conhecida.
- **Python não no PATH**: instalado via winget, PATH permanente. Documentado na skill.
- **PowerShell + strings Python com aspas**: texto longo vai via arquivo `.py`, nunca inline no `-c`. Já era regra da skill.
- **Tabelas PascalCase no Postgres**: sempre usar aspas duplas. Documentado na skill.
