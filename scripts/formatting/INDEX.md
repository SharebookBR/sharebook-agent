# Scripts de Formatação

Utilitários de formatação para mensagens e apresentação.

## Scripts
- `format_two_arrays_whatsapp.py` — formata duas colunas com pontilhado para WhatsApp. Default operacional `max_chars=28`.

## Uso
- Para quadros numéricos e status alinhados no WhatsApp, usar este script em vez de alinhar manualmente.
