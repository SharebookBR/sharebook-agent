# Scripts de Formatação

Utilitários de formatação para mensagens e apresentação.

## Scripts
- `format_two_arrays_whatsapp.py` — formata duas colunas com pontilhado para WhatsApp. Default operacional `max_chars=28`.

## Preferências de Formatação (WhatsApp/Telegram)
- Para quadros numéricos, status operacionais e listas alinhadas, preferir **bloco de código** (monoespaçado).
- Para textos grandes, o script acima pode falhar; manter a concisão brutal em blocos de código.
- Formato preferencial:
  ```text
  Tarefa 1 ............ done
  Tarefa 2 ............ todo
  Tarefa 3 ........ canceled
  ```

## Uso
- Para quadros numéricos e status alinhados no WhatsApp, usar este script em vez de alinhar manualmente.
