Fundamentos de Node.js em 30 minutos

1) O que é Node.js
Node.js é um ambiente que permite executar JavaScript fora do navegador.
Ele é muito usado para APIs, automações e serviços web.

2) Setup inicial
- Instale Node LTS
- Verifique com: node -v e npm -v
- Crie o projeto: npm init -y

3) Primeiro script
Crie index.js:
console.log('Olá, Sharebook!');
Execute: node index.js

4) API mínima
Instale Express:
npm i express

Código base:
const express = require('express');
const app = express();
app.get('/', (req,res)=>res.send('ok'));
app.listen(3000);

5) Próximos passos
- Aprender rotas e middlewares
- Validar entrada de dados
- Conectar com banco
- Fazer deploy
